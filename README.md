# Sample_Pericia
Computer Forensics & Judicial Expertise

Este repositório centraliza metodologias, modelos de documentos e ferramentas utilizadas em minhas atuações como Perita Judicial e Assistente Técnica Forense. O objetivo é manter um padrão de excelência técnica, garantindo a integridade da cadeia de custódia e a clareza científica nos laudos apresentados ao Judiciário.
